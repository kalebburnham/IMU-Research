i.MX Applications Processors based on ARM® Cores

i.MX applications processors are multicore ARM®-based solutions for multimedia and display applications with scalable, high performance, and low power capabilities.

i.MX 8 Series

  • 64-bit ARM®v8-A applications
  • processors transforming
  • interactions in ways
  • you’ve never imagined
i.MX 6 processors based on ARM® Cortex-A9, Cortex-A9 + Cortex-M4 or Cortex-A7 cores
  • Scalable and vivid graphics
  • Outstanding video performance
  • Feature and performance scalability
  • Clock speeds up to 1.2 GHz
  • Remarkable user experience
More Information

i.MX 7 Series

  • Low power applications processors for IoT applications
i.MX 8 Family processors based on ARM® Cortex-A53, Cortex-A72 + Cortex-M4 cores
  • Multi-screen (4x) support
  • Hardware-based Virtualization with Secure Domain Isolation
  • Split-GPU Graphics architecture
  • SafeAssure Failover-capable Display
  • Integrated vision processing
  • Automotive, Industrial & Consumer
More Information

i.MX 6 Series

  • General-purpose applications processors for automotive, consumer, and industrial
i.MX 7 processors based on ARM® Cortex-A7 + Cortex-M4 cores
  • Real Time Responsiveness
  • Heterogeneous Multi-core Processing
  • Multi-level memory system
  • Advanced Security
  • Interface Flexibility
